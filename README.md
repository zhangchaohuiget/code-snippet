## code snippet 代码片段

​		本项目是对开发过程中常用的技术做验证。通过Spring Boot快速构建应用并集成各种技术，创建可独立运行的项目，对技术做验证。

​		每项技术的验证均为一个独立的项目，采用最小依赖，可作为参考将其引入到自己实际的项目中去使用。

​		后续将补充每项技术的讲解、经验，请关注个人博客：www.javadebug.com
